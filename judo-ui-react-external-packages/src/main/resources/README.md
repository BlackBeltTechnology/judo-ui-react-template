# judo-tatami-ui-packages

Re-packaged external dependencies which do not provide UMD modules.

## Items
- @mui/x-data-grid
- @mui/x-date-pickers
- @mui/x-date-pickers/AdapterDayjs
- react-dom/client
- react/jsx-runtime

## Setup

```bash
pnpm i
```

## Commands

```bash
pnpm build

pnpm watch
pnpm preview
```
